Python Projects

As part of my 2nd year college studies in fall 2022, I began learning programming. This started with python, a very simple programming language with many avenues and uses --many of which are showcased through the significantly longer playlist on the channel.


Triangle Solver

At some point during the semester that I began learning python in, I had a trigonometry test coming up.This inspired me as I had recently learned about the math functions in python and set out with my new knowledge to create a tool to solve triangles given three pieces of information. The program can either use the Law of Sines or Law of Cosines to solve the triangle depending on the information it is fed. For one of my first programs that had a real use, I was quite proud of it back then. Perhaps one day a more refined version in another language will hit the channel?





How to Ask a Girl Out With Code

Not too long (about 3 days) after the Triangle solver, I met a girl that I crushed on in a Biology class. She isn't relevant aside from being the inspiration for this project -- A literal representation of how to ask a girl out. As someone who was quite shy (at the time) around women, this was a manifestation of that. As one of my most viewed videos on the channel, it certainly brings a smile to my face realizing just how simple of a program it really was. Eventually I went in with JavaScript Databases and made a matchmaking program as a nod to this very video. I even reference it in the introduction (as a gag)!



Data Analysis - Birthdays

This is in the next semester after the lat three projects, that girl has ghosted me, and I was eager to continue programming. I used the newly acquired File IO (input/output) skills to create a program that tracked the ages of people given their birthdays in a file. This project would eventually lead into a fascination and almost knack for databasing. It has been recreated in so many formats even in a Unity Application, however this humble python program is the original!



Checkers (Parts I & II)

A month after the last project, I posted two videos: Checkers Parts 1 and 2. I used turtle graphics to recreate checkers which at the time was impressive considering I did not really use many functions so the code ended up being a long 18,000 line affair. As an aspiring game developer this was just the start of recreating games with python without using the pygame or similar modules. Overall this feat gave me more confidence to push forward learning programming, especially at a time where learning Objects & Classes in Java was blowing my mind.






Chess

Now we are getting to the big projects, fast-forward from the last project. I have done a few more and now it is summertime, finished my first Java class, and learned C++ and C# on my own. What do I do? Hang out with friends, hahaha don't make me laugh. I made Chess in 48,000 lines of code instead! Similar to checkers I did not use any pygame or similar modules, drawing everything with the turtle graphics module. This was also my first time using Objects and Classes in python, which really validated their usefulness to me. This led me to set my bar even higher after achieving this feat: Monopoly.

Custom MIPS Assembler

As of creating this website, my most recent python project, and likely my most ambitious 'for-fun' project. At this point I had done a database related project in Python, Java, C#, C++, C, Rust, PHP, JavaScript, and Haskell (never shown since I am still learning the language overall). But there were no database tools in Assembly. So as a clinically insane programmer, what did I do? I made a python program to take all my assembly programs and run them just like my Asm IDE (MARS), then add custom instructions for database operations and make an assembly program to utilize them. Cool part is, this one took a little over 3,00 lines of python code to achieve. The video ended up being nearly an hour long as I wanted to show off all the assembly programs I wrote in a previous semester working through it, as well as the custom program I wrote to do databasing. I am extremely proud of this program, and it certainly blew a lot of my classmates' minds!
